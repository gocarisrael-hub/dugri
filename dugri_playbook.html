2 רקע לבן + מידות (עיבוד אחרי ChatGPT)

אחרי שמורידים PNG מ-ChatGPT: מלבינים את הרקע והופכים ל-webp במידה הנכונה, עם ImageMagick (magick). לא צריך כלי AI.

רקע לבן — flood fill מהפינות (fuzz 14%)

fuzz 14% בטוח למוצרים בהירים; 18% כבר "בורח" פנימה. תמונות הקרוסלה/כרטיס נשמרות ב- site/assets/designs/<id>/cover.webp. 

magick input.png \
  -fuzz 14% -fill white \
  -draw "color 0,0 floodfill" \
  -draw "color %[fx:w-1],0 floodfill" \
  -draw "color 0,%[fx:h-1] floodfill" \
  -draw "color %[fx:w-1],%[fx:h-1] floodfill" \
  -resize 900x -quality 86 \
  site/assets/designs/<id>/cover.webp

תמונות גלריה חדות לעמוד המוצר (~1100px)

magick -density 200 -background white input.png \
  -resize 1100x -quality 88 \
  site/assets/designs/<id>/gallery-front.webp

תמונות הירו (רוחב מלא, ~1600px)

magick input.png -resize 1600x -quality 82 site/assets/hero-1.jpg

3 רשימת מילים → CSV → Bulk Create (Canva)

הליבה של כל הזמנה: "רשימת מילים פנימה, PDF החוצה".

  1. ממירים את רשימת המילים ל-CSV (הסקריפט למטה).
  2. ב-Canva מחברים את שדות הטקסט לתוויות הנתונים (autofill) לפי כותרות ה-CSV.
  3. Bulk Create מייצר דף לכל שורה.
  4. מייצאים PDF.
32 עמודות c1w1…c8w4 (c=קלף, w=מילה). כל שורה = דף שלם של 8 קלפים × 4 מילים. מסירים כפילויות, מערבבים, ומשלימים את השורה האחרונה במחרוזות ריקות עד 32. הלוח נשאר עותק יחיד — לא חלק מה-Bulk Create.
תבנית קיימת: alias Shira · Canva design id DAHML5i7T6k.

הסקריפט (Python)

import csv, random, math

def build_csv(words, out, shuffle=True, seed=42):
    seen = set()
    uniq = [w for w in (x.strip() for x in words)
            if w and not (w in seen or seen.add(w))]
    if shuffle:
        random.seed(seed); random.shuffle(uniq)
    PER = 32
    pages = math.ceil(len(uniq) / PER)
    padded = uniq + [''] * (pages * PER - len(uniq))
    headers = [f"c{c}w{w}" for c in range(1, 9) for w in range(1, 5)]
    with open(out, 'w', encoding='utf-8', newline='') as f:
        wr = csv.writer(f); wr.writerow(headers)
        for p in range(pages):
            wr.writerow(padded[p * PER:(p + 1) * PER])

4 הוראות הדפסה וגזירה

  • קלפים: נייר 300 גרם, צבע.
  • קריטי: להדפיס בגודל מקורי 100% — לא "התאם לעמוד" (אחרת הקלפים לא יתאימו לגזירה).
  • כולל קווי גזירה + הוראות גזירה בקובץ.
  • אספקה: הקובץ נשלח עד 24 שעות מרגע קבלת רשימת המילים (לא מרגע התשלום).
